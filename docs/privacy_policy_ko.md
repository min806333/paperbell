# 개인정보처리방침

최종 업데이트: 2026년 3월 27일

## 1. 앱과 연락처

이 개인정보처리방침은 `PaperBell` 앱에 적용됩니다.

- 기본 앱 이름: PaperBell
- 기본 Play 제목 권장안: PaperBell - Doc Reminder
- 한국어 스토어 표기: 담아알림 - 문서 리마인더
- 문의 이메일: `lup53699@gmail.com`

PaperBell은 이미지, PDF, 안내문 같은 문서를 바탕으로 알림을 정리하는 로컬 우선 앱입니다.

## 2. 기본 원칙

PaperBell은 다음 원칙을 바탕으로 동작합니다.

- 로그인 없이 사용할 수 있습니다
- 별도 백엔드 서버를 운영하지 않습니다
- 인증 기능이 없습니다
- 클라우드 동기화가 없습니다
- 핵심 문서 및 리마인더 데이터는 기기 내부에 저장됩니다

## 3. 앱이 다루는 데이터

앱은 사용자가 직접 가져온 문서와, 그 문서에서 정리한 리마인더 정보를 기기 내부에서 다룹니다.

예:

- 가져온 이미지 파일
- 가져온 PDF 파일
- OCR 또는 PDF 읽기를 통해 얻은 추출 결과
- 사용자가 수정하거나 저장한 제목, 날짜, 금액, 메모, 리마인더 정보

이 데이터는 기본적으로 사용자의 기기 내부에만 저장되며, 앱 자체의 서버로 전송되지 않습니다.

## 4. 앱이 하지 않는 일

PaperBell은 현재 다음과 같은 방식으로 동작하지 않습니다.

- 사용자 계정 생성
- 로그인 요구
- 자체 서버로 문서나 추출 결과 전송
- 클라우드 백업 또는 클라우드 동기화
- 광고 네트워크를 통한 개인 데이터 수집

## 5. 후원 결제 안내

앱에는 선택적 1회성 후원 기능이 포함될 수 있습니다.

- 후원 결제는 Google Play 또는 Apple App Store 같은 앱 마켓의 결제 시스템을 통해 처리될 수 있습니다
- 앱 자체는 사용자의 전체 결제 수단 정보를 직접 보관하지 않습니다
- 결제 처리와 영수증, 거래 기록 보관 방식은 각 앱 마켓 사업자의 정책을 따를 수 있습니다

## 6. 문의 이메일과 자발적 전달 정보

사용자가 문의 메일을 보내는 경우, 사용자가 메일 본문에 직접 적은 내용은 이메일을 통해 전달됩니다.

즉:

- 앱 내부 로컬 데이터 처리와는 별개로
- 사용자가 자발적으로 보낸 이메일 내용은 개발자가 확인할 수 있습니다

문의 시에는 필요한 정보만 보내주시고, 민감한 개인정보는 신중하게 다뤄주세요.

## 7. 보관 기간과 삭제

앱 내부 데이터는 원칙적으로 다음 시점까지 기기에 남을 수 있습니다.

- 사용자가 앱에서 직접 데이터를 삭제할 때까지
- 사용자가 앱을 삭제할 때까지
- 기기 백업 또는 운영체제 정책에 따라 사용자의 기기 환경 안에서 유지될 때까지

삭제 방법:

- 앱 내에서 리마인더 또는 로컬 데이터를 삭제할 수 있습니다
- 앱을 삭제하면 기기 내부의 앱 데이터도 함께 제거될 수 있습니다

단, 앱 마켓 결제 기록이나 사용자가 보낸 이메일은 각 서비스 제공자의 정책에 따라 별도로 보관될 수 있습니다.

## 8. 사용자 주의 사항

사용자가 직접 가져오는 문서에는 개인정보가 포함될 수 있습니다.

예:

- 이름
- 주소
- 납부 정보
- 계약 정보

따라서 기기 잠금, 백업 관리, 화면 공유 주의 등 기본적인 기기 보안 관리를 권장합니다.

## 9. 변경 사항

앱 기능이나 데이터 처리 방식이 바뀌면 이 문서를 함께 업데이트할 수 있습니다.

중요한 변경이 있을 경우, 스토어 정보나 앱 내 안내 문구도 함께 정리하는 것을 원칙으로 합니다.
