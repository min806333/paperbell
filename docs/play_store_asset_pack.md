- Global app name: `PaperBell`
- Recommended default Play title: `PaperBell - Doc Reminder`

- Tone:

  - trustworthy
  - local-first
  - privacy-first

  - generic scanner-app styling

## Icon Concept

- A clean paper or document card is the main shape
- A small clock cue communicates reminder behavior without looking like finance or payment software
- Soft green-gray and warm sand tones keep the app quiet and reassuring
- The core message is: keep the paper, remember it later

## Feature Graphic Concept

- Global default wordmark: `PaperBell`
- Keep the current icon direction and overall composition
- Large icon on the left, generous breathing room, wordmark on the right
- Supporting badges:
  - local-first
  - privacy-first
- Do not add screenshots into the feature graphic

If a Korean localized feature graphic is needed later, duplicate the current SVG and replace only the wordmark and subtitle text. Keep the icon, spacing, and composition unchanged.
