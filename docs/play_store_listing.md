## Full Description

### English

PaperBell is a local-first app that helps you bring in photos, screenshots, and PDFs, pull out the dates and amounts that matter, and turn them into calm reminders.

It is designed to feel quiet and trustworthy rather than financial or scanner-like. The focus is simple: help you keep everyday notices, bills, contracts, and document-based tasks from slipping past you.

Key features:

- Import from camera, gallery, PDF, or shared documents
- Review the document before OCR or PDF text reading
- Check and edit dates, amounts, and currency
- Save reminders locally and revisit them later
- Browse reminders in a list or a lightweight calendar view

Privacy and data:

- No login
- No backend server
- No cloud sync
- Imported images, PDFs, extracted results, and reminder data stay on the device

Support:

- PaperBell is provided without ads
- If you wish, you can make a one-time support purchase through the app store billing system
- All core features remain available without support purchases
